# InvFlow 进销存管理系统 - 业务指南

## 采购流程
1. 进入"采购管理"页面，点击"新增采购单"
2. 填写采购单号、选择供应商、仓库
3. 添加采购明细：选择商品、填写数量和单价
4. 保存后状态为"草稿"
5. 点击"入库"完成入库操作，库存自动增加

## 销售流程
1. 进入"销售管理"页面，点击"新增销售单"
2. 填写销售单号、选择客户、仓库
3. 添加销售明细：选择商品、填写数量和售价
4. 保存后状态为"草稿"
5. 点击"出库"完成出库操作，库存自动扣减（按FIFO先进先出）

## FIFO先进先出法
- 每次采购入库时，系统在BatchStock表记录入库批次（含单价）
- 销售出库时，按入库时间从最早的批次开始扣减
- 例如：1月进货10元/件，2月进货12元/件，3月卖出8件，从1月批次扣减
- 这样早期低价货先出库，剩余的是高价库存

## 调拨流程
1. 进入"调拨管理"页面，点击"新增调拨单"
2. 选择调出仓库和调入仓库
3. 添加调拨明细：选择商品、填写数量
4. 保存后状态为"草稿"
5. 点击"审核"完成调拨，调出仓扣减、调入仓增加

## 采购退货流程
1. 进入"采购退货"页面，点击"新增退货单"
2. 选择原采购单，系统自动加载明细
3. 填写退货数量（不能超过原采购数量）
4. 保存后点击"执行退货"，系统按原采购批次扣减库存

## 销售退货流程
1. 进入"销售退货"页面，点击"新增退货单"
2. 选择原销售单，系统自动加载明细
3. 填写退货数量
4. 保存后点击"执行退货"，系统按原出库批次恢复库存

## 盘点流程
1. 进入"盘点管理"页面，点击"新增盘点单"
2. 选择仓库，点击"加载全部商品"自动录入所有商品
3. 核对实盘数量，系统自动计算差异
4. 点击"审核"，盘盈盘亏自动生成损益单调整库存

## 查看报表
- 月度明细账：查看某月所有商品的出入库流水
- 销售毛利统计：查看均价法下的毛利分析
- FIFO销售毛利：对比FIFO与均价法的毛利差异
- FIFO成本对比：对比两种成本算法的差异
